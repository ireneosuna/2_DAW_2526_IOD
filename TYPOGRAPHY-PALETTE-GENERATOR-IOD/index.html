Typography Palette Generator

Descripción del proyecto

Este proyecto tiene como objetivo crear una herramienta que nos permita experimentar con paletas de colores y tipografías para comprender cómo afectan al diseño web.

Elementos de prueba

  • Párrafos de texto
  • Listas con viñetas
  • Enlaces externos
  • Imágenes
  • Formularios

Puedes consultar más información sobre el uso del color en diseño en Wikipedia.
